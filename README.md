# いいね～む

service: http://www.iiname.net  
source:  https://github.com/iine/iiname

## Description

ユニークなチーム名でチームの個性をアピールしたい  
ハッカソンやグループワークなどのメンバー向けの  
チーム名ブレインストーミングツール  
**いいね～む**です。  
短時間でどんどんアイデアを提案することができ、  
「チームＡ」のような名前で済ませるのと違って、  
チームビルディング初期のアイスブレイクの効果が備わっています。
